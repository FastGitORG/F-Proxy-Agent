# SNIProxyGo
Simple SNI proxy written in go

## Compile
* go get gopkg.in/yaml.v2
* go build

## Usage:
* Open port 443
* Edit rules in config.yaml
* Start

## Usage:
Usage of ./SNIProxyGo:  
  -D    Enable debug  
  -F string   
        log to file  
  -c string  
        config file (default "config.yaml")
