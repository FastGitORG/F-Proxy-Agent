# SNIProxyGo
一个简单的SNI代理实现

## README in English: [Here](https://github.com/TachibanaSuzume/SNIProxyGo/blob/master/READMEEN.md "link"). 

## 编译
* go get gopkg.in/yaml.v2
* go build

## 使用方式:
* 打开443端口的访问权限
* 修改config.yaml中的规则
* 运行

## 命令行附加参数
*  -D 允许调试输出
*  -F log.txt 记录输出到文件
*  -c config.yaml 使用特定的config文件
